img 태그

    - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

    [속성]
    (속성 : 시작 태그에 작성하는 내용)

    1. src : 삽입할 이미지의 경로를 작성하는 속성
            (파일 위치, 인터넷 주소)
    
    2. width / height : 너비 / 높이를 지정하는 속성

    3. alt (대안) : 이미지 설명을 작성하는 속성
            -> 이미지가 출력되지 않은 경우 화면에 출력됨
            + 웹 접근성(스크린 리더가 이미지 설명을 읽어줌)

img 태그 예시

웹 이미지 얻어와 출력

이미지 크기 지정

단위에 따른 크기 변화

고정 크기 단위(px)화면 크기가 변해도 항상 고정된 크기를 지님

가변크기 단위(%)화면 또는 영역에 대한 상대적인 크기 단위

고정 크기 + 가변 크기

  고정 크기로 설정된 요소 (태그) 내부에서
  일정한 크기의 이미지 또는 요소를 출력하고 싶을 떄 
  가변 크기 단위(%) 이용해서 지정
