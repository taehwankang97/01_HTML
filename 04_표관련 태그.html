표 관련 태그

  1. table(표)태그

            -웹 문서에서 자료를 정리하기위한 태그
        - table 태그는 행, 열을 작성하기 위한판을 지칭하는 태그

  2. tr태그

     
        - 테이블의 한 행을 나타내는 태그

  3. td(table Data)태그

              -테이블 데이터를 작성할 수 있는 태그
          -각 행(tr)의 열 (column) 역활을함
          -엑셀의 한 셀(한 칸)

  4. th(table Header) 태그 

    테이블의 제목
          또는 내용을 강조할 때
          td 대신 사용

          -굵은 글씨 + 가운데 정렬

  5. caption 태그

            -테이블에 대한 설명을 작성하는 태그

  6. thead 태그

            - 테이블 상단 영역(컬럼 제목 영역)

  7. tbody 태그

            - 테이블 중단영역(값 작성 영역)

  8. tfoot 태그

          - 테이블 하단 영역 (합계, 총합 영역)
